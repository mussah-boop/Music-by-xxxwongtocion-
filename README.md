🎶 FGH Music Player

Karibu kwenye *FGH Music Player*!  
Hii ni app rahisi ya kuchezesha nyimbo (audio player) iliyojengwa kwa kutumia *HTML*, *CSS*, na *JavaScript*.

📂 Files
- `index.html` — kurasa kuu ya player.
- `style.css` — muonekano (design) ya player.
- `script.js` — kusimamia tabia za player (optionally).
- `music.mp3` — Faili la muziki linalochezwa.

🚀 Jinsi ya kutumia
1. Fungua `index.html` kwenye browser yako.
2. Hakikisha faili ya muziki (`music.mp3`) iko kwenye folda moja na HTML.
3. Furahia kusikiliza muziki wako! 🎧

📌 Features
- Play/Pause nyimbo
- Simple and clean user interface
- Lightweight and fast to load

🛠️ Technologies
- HTML5
- CSS3
- JavaScript (optional kwa features zaidi)
